# Vue-Builder 2.0.0

>这个是一个定制版的模版

## 技术栈
- [X] vue
- [X] vuex
- [X] vue-router
- [X] vue-resource
- [X] vuex-persistedstate
- [X] immutable
- [X] webpack
- [X] eslint
- [X] babel

## Install

```
# 如果已经安装了可以忽视
npm install vue-cli -g

# 初始化模版
vue init Timson020/Vue-Builder ProjectName
```

## Usage
```
# install the package
npm install

# dev
npm run dev

# release
npm run build

# create the router page
npm run r LogIn

# create the global component
npm run c Button
```
